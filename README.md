## Description

Backend for a food delivery platform

Technologies used: 
Language: Node Js - nestjs framework
Database: Mysql 
Api Tool: Swagger

## Installation

```bash
$ npm install
```

SET restaurant & user data url in .env file.
RESTAURANT_DATA_URL=
USER_DATA_URL=

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
